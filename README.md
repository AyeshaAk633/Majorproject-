# Corporate Website Development

## Project Overview
This project is a Corporate Digital Presence Platform developed for **Outpro.India**. The website is designed to provide a modern, responsive, and scalable online presence that showcases the company's services, portfolio, team, and business capabilities. It focuses on improving brand credibility, supporting lead generation, and delivering a seamless user experience across devices.

## Features
- Home Page
- About Us
- Services
- Portfolio / Case Studies
- Client Testimonials
- Contact Form with MongoDB Database
- Responsive Design (Mobile, Tablet & Desktop)

## Tech Stack

## Installation

### Backend

```bash
cd Majorproject/backend
npm init -y
npm install express mongoose cors dotenv
node server.js
```

### Frontend

```bash
cd Majorproject/frontend
npm create vite@latest
npm install
npm run dev
```
